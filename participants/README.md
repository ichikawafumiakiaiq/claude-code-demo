# 参加者ページの作り方

このディレクトリは、受講者の皆さんが **自分の好きなページを作る場所** です。

## 手順（全部 Claude に日本語で頼めばOK）

1. `_template.html` をコピーして `<あなたの名前>.html` を作る
   - 例：`taro.html`、`hanako.html`、`ichikawa.html` など
2. Claude に内容を頼む（自己紹介、お気に入り、なんでも）
3. `../index.html` の参加者リンク一覧に自分のリンクを1行追加
4. commit & push → PR 作成

## やってOK

- 自己紹介ページ、お気に入り紹介、ミニ作品、おすすめ、なんでも
- ページ内のデザインや構成は自由
- 画像URL（ネット上にあるもの）を貼ってもOK

## やらないでください

- 他の人のファイル（`<他人の名前>.html`）の編集
- 機密情報・個人情報の記載
- 公開して困るものの記載（公開リポジトリです）
- `/demo/` 配下のファイル変更（発表者用）

## 動作確認

作ったHTMLファイルを Finder でダブルクリックすれば、ブラウザで開いて見えます。
サーバーを起動する必要はありません。

## 困ったら Claude に

- 「いま何が起きてる？」「専門用語を使わずに説明して」
- 「さっきの変更を取り消して、元に戻して」
- 「commit を取り消して」

日本語で聞けば全部 Claude が答えてくれます。
